# go-vpn

go-vpn is a CLI written in Go.

This CLI is intended for simplying the process of connecting to VPNs on MacOS that require Google Authenticator.

## Requirements
- Golang 1.14
- Oathtool

## Setup

1. Install Oath
```brew install oath-toolkit```

## Usage

1. Configure

`go-vpn configure airbnb pass123`

`go-vpn getOtp gojek`

go-vpn connect int
